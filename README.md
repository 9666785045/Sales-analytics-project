# Sales Analytics Dashboard – Tableau Project
## Project objective
This repository contains a fully-designed Sales Analytics Dashboard created using Tableau Public.
The project demonstrates skills in data cleaning, analysis, and visual storytelling using interactive dashboards.

##Dataset used
- <a href="https://github.com/9666785045/Sales-analytics-project/blob/main/data%20(sample-data-10mins)_data.csv">Dataset</a>

## Questions (KPIs)
-1. Total Sales Amount
Overall revenue generated from all countries.
Helps evaluate financial performance.
-2. Total Boxes Shipped
Measures product movement and overall demand.
Useful for logistics and supply chain planning.
-3. Total Number of Records / Transactions
Indicates dataset volume.
Reflects business activity level.
-4. Sales by Country
Identifies the highest-revenue countries.
Helps compare region-wise performance.
-5. Top 15 Salespersons Performance
Ranks sales executives based on amount sold.
Helps in talent evaluation and rewards planning.
-6. Average Sales per Salesperson
Measures productivity and efficiency.
-7. Contribution % by Country
Shows each country’s share in total revenue.
