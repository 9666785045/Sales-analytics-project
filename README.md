# Sales Analytics Dashboard – Tableau Project
## Project objective
This repository contains a fully-designed Sales Analytics Dashboard created using Tableau Public.
The project demonstrates skills in data cleaning, analysis, and visual storytelling using interactive dashboards.

##Dataset used
- <a href="https://github.com/9666785045/Sales-analytics-project/blob/main/data%20(sample-data-10mins)_data.csv">Dataset</a>

## Questions (KPIs)
-1. Total Sales Amount

-2. Total Boxes Shipped

-3. Total Number of Records / Transactions

-4. Sales by Country

-5. Top 15 Salespersons Performance

-6. Average Sales per Salesperson

-7. Contribution % by Country

## Dashboard Features
-1: KPI Summary
Shows a high-level snapshot of:
Total Revenue
Total Boxes Shipped
Data Count

-2️: Sales by Country – Pie Chart
Visual comparison of regional sales distribution.

-3️:Top 15 Salespersons – Horizontal Bar Chart
Ranks representatives based on sales performance.

-4️:Interactive Filters
Allows viewers to explore patterns instantly.
